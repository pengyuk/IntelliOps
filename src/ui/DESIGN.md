# 前端设计（UI）

## 目标

提供直观的事故看板与现场排查体验，降低沟通成本并支持多角色并行协作。

## 关键页面与组件

- 事故看板：全局态势、未处理事件、SLA 风险
- 事件详情页：现场信息卡片、时间线、相关变更与日志片段
- 排查助手：推荐步骤、可执行操作、日志检索入口
- 协同面板：成员列表、实时消息、会议记录、屏幕共享入口
- 复盘管理：复盘任务、整改状态、知识条目链接

## 交互要点

- 所有关键动作要有明确审计与回退路径
- 支持多人实时更新（基于 WebSocket）
- 卡片化信息以便快速浏览与操作

## 技术选型

- React + AntD 或 Vue + Element
- 状态管理：Redux / Vuex
- 通信：WebSocket + REST API
